## 📌 What is this?

Habit Tracker is a web application for tracking daily habits and personal goals.

---

## 🛠 What was used?

- Python
- Django
- PostgreSQL
- HTML
- CSS

---

## ▶ How to run?

```bash
git clone https://github.com/marekxsr/habit-tracker-django.git

cd habit-tracker-django

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

After running the server, open your localhost in the browser.
